# Visão Geral

Este portfólio é um site pessoal e profissional em HTML5, CSS3 e JavaScript puro. O objetivo é exibir projetos, habilidades e contato de forma objetiva, responsiva e sem dependências externas.

A interface estática inclui navegação por seções, suporte a tema claro/escuro e internacionalização com JSON. A experiência é construída para ser leve, fácil de manter e compatível com desktop e mobile.

As tecnologias principais são HTML5, CSS3 e JavaScript puro, com JSON usado para tradução. Isso garante uma implementação limpa, focada em front-end sem build tools.

O objetivo final é validar domínio de front-end, apresentar trabalhos e fornecer um canal profissional de contato de forma clara e técnica.

# Início Rápido

Não há instalação de dependências ou build.

## Execução local

- Abra `Index.html` diretamente no navegador.
- Ou use Live Server no VS Code para desenvolvimento com recarga automática.

## Edição rápida

- Textos fixos: edite `Index.html`.
- Tradução: edite `pt.json` e `en.json`.
- Comportamento: edite `script.js`.
- Estilo global: edite `base.css`.
- Layout e responsividade: edite `layout.css`.

## Validação

- Verifique troca de idioma.
- Verifique tema claro/escuro.
- Verifique abertura e fechamento de modais.
- Confirme que imagens e links funcionam.
