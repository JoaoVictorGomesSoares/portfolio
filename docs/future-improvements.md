# Future Improvements

A primeira melhoria planejada é a adoção de testes automatizados com Jest e Cypress. Eles ajudam a validar a lógica de JavaScript puro, a navegação e o comportamento dos modais.

A otimização de imagens com WebP e lazy loading é outra prioridade. Isso melhora a performance, reduz o peso inicial e garante carregamento mais rápido em dispositivos móveis.

A internacionalização (i18n) deve evoluir para um fluxo mais robusto. Atualmente a tradução usa JSON estático, e esse modelo pode ser expandido para suportar novos idiomas sem impactar a base de código.

Adicionalmente, é recomendada a introdução de lint e build leve para CSS e JavaScript, preservando a simplicidade enquanto aumenta a qualidade do código.
