# Início Rápido

Este projeto é um site estático. Não exige instalação de dependências nem processo de build.

## Requisitos

- Navegador moderno
- Editor de código, como VS Code

## Executando localmente

### Opção 1: abertura direta

1. Navegue até a pasta do projeto.
2. Abra o arquivo `Index.html` no navegador.

Essa opção funciona bem para visualização rápida.

### Opção 2: servidor local

1. Abra a pasta do projeto no VS Code.
2. Use uma extensão como Live Server.
3. Rode a página a partir da raiz do projeto.

Essa opção é melhor para desenvolvimento, porque facilita recarga e inspeção.

## Primeiras edições comuns

- Atualizar textos fixos: edite `Index.html`.
- Atualizar traduções: edite `pt.json` e `en.json`.
- Atualizar comportamento: edite `script.js`.
- Ajustar aparência global: edite `base.css`.
- Ajustar layout e responsividade: edite `layout.css`.

## Validação rápida

Depois de qualquer mudança, confira:

- Troca de tema funcionando.
- Botão de idioma alternando corretamente entre `pt-BR` e `en-US`.
- Modais abrindo e fechando sem travar a página.
- Links principais e botões de currículo funcionando.
- Seções responsivas em desktop e mobile.
